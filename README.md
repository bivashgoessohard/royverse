# Royverse Website

This is a simple blog and product review website for the domain `royverse.in`.

## How to Update Content

The content of the website is stored in HTML files. To update the content, you can edit the following files:

- `index.html`: The home page.
- `about.html`: The about page.
- `contact.html`: The contact page.
- `privacy.html`: The privacy policy.
- `blog/post1.html`: The first blog post.
- `blog/post2.html`: The second blog post.

You can add new blog posts by creating new HTML files in the `blog` directory.

## Deployment

This website is configured for deployment on Netlify. To deploy the website, you can connect your GitHub repository to Netlify.

## Domain Setup

After deploying the website, you will need to update the DNS records for `royverse.in` to point to the Netlify servers. The specific DNS records will be provided by Netlify after you've set up your site.
